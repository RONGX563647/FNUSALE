# 用户模块功能设计文档

**版本**: v1.1.1

---

## 一、模块概述

### 1.1 模块定位

用户模块是FNUSALE校园二手交易平台的核心基础模块，承担着平台信任体系构建的关键职责。作为校园专属二手交易平台，用户模块的核心价值在于：

- **信任基础**：通过校园身份认证和定位认证双重验证，确保交易双方均为校内师生
- **安全交易**：建立完善的信誉体系，保障交易安全
- **便捷体验**：提供校园自提点管理，简化交易流程

### 1.2 模块职责

| 职责领域 | 具体内容 |
|---------|---------|
| 用户管理 | 用户注册（手机号/邮箱）、用户登录（手机号/邮箱）、信息管理、密码管理 |
| 身份认证 | 校园卡/学生证审核、学号/工号绑定 |
| 定位认证 | 校园围栏校验、定位权限管理 |
| 地址管理 | 用户地址CRUD、默认地址管理、自提点关联 |
| 自提点管理 | 校园自提点维护、附近自提点查询 |
| 信誉体系 | 信誉分计算、信誉等级管理、权限限制 |
| 签到服务 | 每日签到、连续签到奖励、签到记录 |
| 评价服务 | 用户互评、评价分计算、评价标签管理 |
| 排行榜服务 | 活跃度排行、交易排行、信誉排行 |

### 1.3 模块边界

**包含功能**：
- 用户账号生命周期管理
- 校园身份认证流程
- 定位认证与校园围栏校验
- 用户地址与自提点管理
- 用户信誉分管理
- 用户签到与积分奖励
- 用户评价与评分管理
- 用户排行榜与数据统计

**不包含功能**：
- 商品发布与管理（商品模块）
- 订单交易流程（交易模块）
- IM即时通讯（IM模块）
- 支付对接（交易模块）

```java
// 密码校验正则（6-20位，至少包含字母和数字）
String PASSWORD_REGEX = "^(?=.*[A-Za-z])(?=.*\\d)[A-Za-z\\d]{6,20}$";

// 用户名校验正则（2-20位中英文、数字、下划线）
String USERNAME_REGEX = "^[a-zA-Z0-9_\\u4e00-\\u9fa5]{2,20}$";

// 验证码校验正则（6位数字）
String CAPTCHA_REGEX = "^\\d{6}$";
```

#### 3.3.2 Token设计

**Access Token**：
- 有效期：2小时
- 存储位置：Redis（key: `user:token:{userId}`）
- 用途：接口访问认证

**Refresh Token**：
- 有效期：7天
- 存储位置：Redis（key: `user:refresh:{userId}`）
- 用途：刷新Access Token

**Token载荷**：
```json
{
  "userId": 1,
  "phone": "13800138000",
  "identityType": "STUDENT",
  "authStatus": "AUTH_SUCCESS",
  "iat": 1704067200,
  "exp": 1704074400
}
```

### 3.4 密码管理

#### 3.4.1 密码加密

采用BCrypt算法进行密码加密存储：

```java
// 加密
String encodedPassword = passwordEncoder.encode(rawPassword);

// 验证
boolean matches = passwordEncoder.matches(rawPassword, encodedPassword);
```

#### 3.4.2 密码修改规则

| 规则项 | 描述 |
|--------|------|
| 旧密码验证 | 必须正确输入旧密码才能修改 |
| 新密码要求 | 不能与旧密码相同 |
| 密码强度 | 6-20位，至少包含字母和数字 |
| 修改后处理 | 清除所有Token，需要重新登录 |

#### 4.2.3 围栏校验算法

```java
/**
 * 校验坐标是否在校园围栏内
 * @param longitude 经度
 * @param latitude 纬度
 * @return true-在围栏内，false-不在围栏内
 */
public boolean isInCampusFence(String longitude, String latitude) {
    // 1. 获取校园围栏配置
    String center = systemConfigService.getConfigValue("campus_fence_center");
    int radius = Integer.parseInt(systemConfigService.getConfigValue("campus_fence_radius"));

    // 2. 解析中心点坐标
    String[] centerCoords = center.split(",");
    double centerLng = Double.parseDouble(centerCoords[0]);
    double centerLat = Double.parseDouble(centerCoords[1]);

    // 3. 计算两点距离
    double distance = calculateDistance(
        centerLng, centerLat,
        Double.parseDouble(longitude), Double.parseDouble(latitude)
    );

    // 4. 判断是否在围栏内
    return distance <= radius;
}

/**
 * 计算两点之间的距离（米）
 * 使用Haversine公式
 */
private double calculateDistance(double lng1, double lat1, double lng2, double lat2) {
    double earthRadius = 6371000; // 地球半径（米）
    double dLat = Math.toRadians(lat2 - lat1);
    double dLng = Math.toRadians(lng2 - lng1);
    double a = Math.sin(dLat / 2) * Math.sin(dLat / 2) +
               Math.cos(Math.toRadians(lat1)) * Math.cos(Math.toRadians(lat2)) *
               Math.sin(dLng / 2) * Math.sin(dLng / 2);
    double c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
    return earthRadius * c;
}
```

```java
public List<PickPointVO> getNearbyPickPoints(String longitude, String latitude, Integer distance) {
    // 1. 获取所有启用的自提点
    List<CampusPickPoint> allPoints = pickPointMapper.selectEnabledList();

    // 2. 计算距离并过滤
    return allPoints.stream()
        .map(point -> {
            double dist = calculateDistance(
                Double.parseDouble(longitude), Double.parseDouble(latitude),
                Double.parseDouble(point.getLongitude()), Double.parseDouble(point.getLatitude())
            );
            PickPointVO vo = convertToVO(point);
            vo.setDistance((int) dist);
            return vo;
        })
        .filter(vo -> vo.getDistance() <= distance)
        .sorted(Comparator.comparingInt(PickPointVO::getDistance))
        .collect(Collectors.toList());
}
```

```java
/**
 * 获取连续签到天数
 * @param userId 用户ID
 * @return 连续签到天数
 */
public int getContinuousSignDays(Long userId) {
    LocalDate today = LocalDate.now();
    int days = 0;

    // 从今天往前遍历，直到找到未签到的日期
    for (int i = 0; i < 365; i++) {
        LocalDate date = today.minusDays(i);
        String key = "user:sign:" + userId + ":" + date.format(DateTimeFormatter.BASIC_ISO_DATE);
        if (Boolean.TRUE.equals(redisTemplate.hasKey(key))) {
            days++;
        } else {
            break;
        }
    }
    return days;
}
```

#### 9.5.2 综合评分计算

```java
/**
 * 计算用户综合评分
 * 采用加权平均 + 时间衰减算法
 * @param userId 用户ID
 * @return 综合评分（1-5分，保留2位小数）
 */
public double calculateUserRating(Long userId) {
    // 获取用户所有有效评价
    List<OrderEvaluation> evaluations = evaluationMapper.selectByEvaluatedId(userId);

    if (evaluations.isEmpty()) {
        return 5.0; // 新用户默认5分
    }

    double totalWeight = 0;
    double weightedSum = 0;

    for (OrderEvaluation eval : evaluations) {
        // 时间衰减权重：越近的评价权重越高
        long daysBetween = ChronoUnit.DAYS.between(eval.getCreateTime(), LocalDateTime.now());
        double timeWeight = Math.exp(-daysBetween / 90.0); // 90天衰减周期

        // 综合评分权重
        double score = eval.getScore();

        weightedSum += score * timeWeight;
        totalWeight += timeWeight;
    }

    return Math.round(weightedSum / totalWeight * 100) / 100.0;
}
```

```java
/**
 * 获取用户评价标签统计
 * @param userId 用户ID
 * @return 标签及出现次数
 */
public Map<String, Integer> getTagStatistics(Long userId) {
    List<OrderEvaluation> evaluations = evaluationMapper.selectByEvaluatedId(userId);

    Map<String, Integer> tagCount = new HashMap<>();
    for (OrderEvaluation eval : evaluations) {
        if (StringUtils.isNotBlank(eval.getEvaluationTag())) {
            String[] tags = eval.getEvaluationTag().split(",");
            for (String tag : tags) {
                tagCount.merge(tag.trim(), 1, Integer::sum);
            }
        }
    }

    // 按出现次数排序
    return tagCount.entrySet().stream()
        .sorted(Map.Entry.<String, Integer>comparingByValue().reversed())
        .collect(Collectors.toMap(
            Map.Entry::getKey,
            Map.Entry::getValue,
            (e1, e2) -> e1,
            LinkedHashMap::new
        ));
}
```

### 10.5 活跃度计算规则

#### 10.5.1 活跃度评分公式

```
活跃度分数 = 签到得分 + 浏览得分 + 发布得分 + 交易得分 + 互动得分

签到得分 = 基础签到(5分) + 连续签到奖励(最高+20分)
浏览得分 = 浏览商品数 × 0.1 (每日上限10分)
发布得分 = 发布商品数 × 5 (每日上限50分)
交易得分 = 完成订单数 × 10 (每笔)
互动得分 = 收到消息数 × 0.5 + 收藏数 × 1 (每日上限20分)
```

#### 10.5.2 活跃度计算服务

```java
/**
 * 计算用户日活跃度
 * @param userId 用户ID
 * @param date 日期
 * @return 活跃度分数
 */
public int calculateDailyActivity(Long userId, LocalDate date) {
    int score = 0;

    // 1. 签到得分
    UserSignRecord signRecord = signRecordMapper.selectByUserIdAndDate(userId, date);
    if (signRecord != null) {
        score += 5; // 基础签到分
        if (signRecord.getContinuousDays() >= 7) {
            score += 20;
        } else if (signRecord.getContinuousDays() >= 3) {
            score += 10;
        }
    }

    // 2. 浏览得分（每日上限10分）
    int browseCount = behaviorMapper.countByType(userId, "BROWSE", date);
    score += Math.min(browseCount * 0.1, 10);

    // 3. 发布得分（每日上限50分）
    int publishCount = productMapper.countByUserIdAndDate(userId, date);
    score += Math.min(publishCount * 5, 50);

    // 4. 交易得分
    int tradeCount = orderMapper.countCompletedByUserIdAndDate(userId, date);
    score += tradeCount * 10;

    // 5. 互动得分（每日上限20分）
    int messageCount = imMessageMapper.countByReceiverAndDate(userId, date);
    int collectCount = behaviorMapper.countByType(userId, "COLLECT", date);
    score += Math.min(messageCount * 0.5 + collectCount * 1, 20);

    return score;
}
```

```java
@Scheduled(cron = "0 10 0 * * ?")
public void updateDailyRanking() {
    LocalDate yesterday = LocalDate.now().minusDays(1);

    // 1. 更新活跃度排行
    updateActivityRanking(yesterday);

    // 2. 更新交易排行
    updateTradeRanking(yesterday);

    // 3. 记录排行历史
    saveRankingHistory(yesterday);

    // 4. 发放排行奖励
    distributeRankingRewards(yesterday);
}

/**
 * 更新活跃度排行榜
 */
private void updateActivityRanking(LocalDate date) {
    String key = "rank:activity:daily:" + date.format(DateTimeFormatter.BASIC_ISO_DATE);

    // 获取所有活跃用户
    List<Long> activeUsers = behaviorMapper.selectActiveUsers(date);

    // 计算每个用户的活跃度分数
    for (Long userId : activeUsers) {
        int score = calculateDailyActivity(userId, date);
        redisTemplate.opsForZSet().add(key, userId.toString(), score);
    }

    // 设置过期时间（保留30天）
    redisTemplate.expire(key, 30, TimeUnit.DAYS);
}
```

```java
 * @param date 排行日期
 */
@Transactional
public void distributeRankingRewards(LocalDate date) {
    // 活跃度排行奖励
    String activityKey = "rank:activity:daily:" + date.format(DateTimeFormatter.BASIC_ISO_DATE);
    distributeReward(activityKey, "ACTIVITY", date);

    // 交易排行奖励
    String tradeKey = "rank:trade:daily:" + date.format(DateTimeFormatter.BASIC_ISO_DATE);
    distributeReward(tradeKey, "TRADE", date);
}

/**
 * 发放单项排行奖励
 */
private void distributeReward(String rankKey, String rankType, LocalDate date) {
    // 获取前100名
    Set<ZSetOperations.TypedTuple<String>> top100 = redisTemplate.opsForZSet()
        .reverseRangeWithScores(rankKey, 0, 99);

    if (top100 == null || top100.isEmpty()) {
        return;
    }

    int rank = 1;
    for (ZSetOperations.TypedTuple<String> tuple : top100) {
        Long userId = Long.parseLong(tuple.getValue());
        int reward = calculateRewardByRank(rank, rankType);

        if (reward > 0) {
            // 发放积分奖励
            pointsService.addPoints(userId, reward, "排行榜奖励-" + rankType + "-第" + rank + "名");

            // 记录奖励日志
            RankingRewardLog log = new RankingRewardLog();
            log.setUserId(userId);
            log.setRankType(rankType);
            log.setRank(rank);
            log.setRewardPoints(reward);
            log.setRankDate(date);
            log.setCreateTime(LocalDateTime.now());
            rankingRewardLogMapper.insert(log);
        }
        rank++;
    }
}
```

```java
public class UserExceptionHandler {

    @ExceptionHandler(UserException.class)
    public Result<Void> handleUserException(UserException e) {
        log.error("用户模块异常：{}", e.getMessage());
        return Result.fail(e.getCode(), e.getMessage());
    }

        String message = e.getBindingResult().getFieldErrors().stream()
```
