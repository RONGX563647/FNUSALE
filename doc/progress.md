# FNUSALE校园二手交易平台 - 项目进度文档

## 项目概述
FNUSALE是一个面向校园用户的二手交易平台，采用微服务架构，提供商品发布、交易、即时通讯、AI赋能等功能。

## 技术栈总结

### 后端技术栈
- **框架**: Spring Boot 2.7.x
- **微服务**: Spring Cloud Alibaba
- **数据库**: MySQL 8.0
- **缓存**: Redis
- **消息队列**: RocketMQ
- **搜索**: Elasticsearch
- **对象存储**: 阿里云OSS
- **认证授权**: JWT
- **API文档**: Swagger/Knife4j

### 前端技术栈（待开发）
- **框架**: Vue 3 / React
- **UI组件库**: Element Plus / Ant Design
- **状态管理**: Pinia / Redux
- **路由**: Vue Router / React Router
- **HTTP客户端**: Axios
